# 2026-07-01 讲师介绍与地图跳转行为优化

## 背景

手机端验收反馈：

- 微信打开 GitHub Pages 链接时会出现“非微信官方网页，请确认是否继续访问”的安全提示。
- 点击高德导航时，微信/高德链路也可能出现确认或中转页面。
- 首屏红框区域不应再展示课程宣传海报预览，应直接放讲师个人介绍。

## 判断

- 微信“非微信官方网页”提示是微信客户端对外部域名的安全提示，不由 H5 页面代码控制。当前使用 `github.io` 域名，无法通过改 HTML/CSS/JS 直接关闭。
- 高德地图跳转确认页属于微信内置浏览器和高德 App 唤起链路的安全限制。当前可做的优化是减少新窗口行为；若要彻底免提示，需要改为微信认证域名、公众号/小程序承载，或接入微信开放标签/业务域名等官方链路。

## 修改

- 首屏原课程定档海报预览替换为讲师介绍卡。
- 讲师介绍内容：
  - 狼格拉底 / 赵博
  - 意心会创始人、AI 管理艺术家、3 家 AI 原生基因公司创始人
  - 2023、23-25、2025.04 三段经历说明
- 移除首屏旧海报预览相关 CSS 和渲染。
- 高德导航链接移除 `target="_blank"`，改为当前 WebView 内跳转，减少另开外部页面行为。
- 同步更新 `index.html` 与中文副本，并重新导出长图。

## 验证

- Playwright 检查 375 / 390 / 430 / 520 宽度均无横向溢出。
- 首屏无旧海报预览，讲师名称显示为“狼格拉底”。
- 页面内 `a[target="_blank"]` 数量为 0。
- 长图重新导出：逻辑宽度 520，高度 6080；实际 PNG 为 1040 × 12160。
