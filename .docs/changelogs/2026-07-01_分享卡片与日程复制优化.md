# 2026-07-01 分享卡片与日程复制优化

## 背景

微信转发 H5 链接时，卡片右侧缩略图为空白；住宿推荐的“复制地址”在高德搜索里不如复制酒店官方门店名稳定；课程日程信息只需展示时间，不需要占用过多纵向空间。

## 修改

- 新增 `share-logo.png`，使用意心会公司 logo 作为微信/社交分享缩略图。
- 在 HTML 静态头部增加分享元信息：
  - `og:title`：老板AI管理工作坊 / 第四期课前须知 / 7月10日～12日。
  - `og:image`：线上 `share-logo.png`。
  - `description`、`twitter:image`、`image_src` 等兜底字段。
- 在 body 顶部增加静态隐藏 logo 图片，兼容不执行 JS 的微信链接预览抓取。
- 住宿推荐按钮由“复制地址”改为“复制酒店名”，复制内容分别为：
  - 重庆壹棠酒店(汽博中心金渝地铁站店)
  - 全季酒店(重庆汽博中心三亚湾地铁站店)
  - 重庆铂驰酒店
- 课程日程从纵向卡片改为三列横向紧凑时间表，仅展示开课、午休、结束时间。
- 在课程日程下方增加问卷与统一教学内容说明。
- 重新导出长图。

## 验证

- Playwright 检查 375 / 390 / 430 / 520 宽度均无横向溢出。
- 390 宽度下课程日程为三列布局。
- `og:title` 解析为三行文本；`og:image` 指向线上 `share-logo.png`。
- 点击全季酒店“复制酒店名”后 toast 显示酒店名复制成功。
- 长图重新导出：逻辑宽度 520，高度 6040；实际 PNG 为 1040 × 12080。
